---- FONTS ----

The following fonts are provided by default:

  TrueType fonts:
    Roboto
    MgOpenModerna

  Vector fonts:
    romans
    scriptc

Additional fonts may be added to the map server by placing them into this
directory (map_data/fonts) and listing them in the fonts section of the 
.sdf file. 

The Google font directory provides a variety of free TrueType fonts that 
can be downloaded from the following location:
  https://fonts.google.com/
