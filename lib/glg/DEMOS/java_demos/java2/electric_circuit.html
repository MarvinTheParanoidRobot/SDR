GLG Java Demo: Electrical Circuit Monitoring

This is an example of an electrical circuits monitoring application running in a browser using the GLG Java Applet. The demo loads a drawing of an electrical circuit and updates it with simulated data. In an application, the data may come from a life process.

The Print Resources to Console button prints all resources that are animated to the Java Console, including a complete resource path for each resource.

The drawing for the demo was created using the GLG Graphics Builder.
