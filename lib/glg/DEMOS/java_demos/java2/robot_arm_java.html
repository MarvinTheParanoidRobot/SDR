GLG Java Robot Arm Demo

This is an example of a simple user-controlled 3D simulation. A simple GLG Java Applet is used to display a robot arm drawing and animate its either with simulated data or with the user input via slider controls. All functionality of this demo is encapsulated in the GLG drawing, while the applet simply displays the drawing and animates it using resources defined in the drawing.

The drawing was created using the GLG Graphics Builder and contains dynamic resources that control the angles of the robot arm's segments. Either a wireframe or a shaded 3D model can be selected at run-time.

User Interaction:

  • Press Stop to turn off automatic update and use sliders to control the robot arm manually.
