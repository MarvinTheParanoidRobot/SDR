GLG 2D and 3D Graphs: Java Demo

This demo demonstrates a few basic 2D and 3D Graphs from the GLG Widgets Catalog.

Examples of real-time charts with integrated scrolling and cursor feedback may be found in the Real-Time Strip-Chart Demo.

Notice: Script buttons may not work in some browsers due to the LiveConnect's new security settings.


This demo applet generates simulated data for updating the graph. In an application, the data can be obtained from any Web data source.

All GLG graphs can be modified using the GLG Graphics Builder with a point and click interface.

The Applet Source Code may be used as an example of using resources of the GLG 2D and 3D graph widgets. The Graph Control Demo shows a more elaborate example of using graph's resources to control the graph's appearance.
