GLG Java Graph Control Demo

This demo demonstrates the use of resources to customize GLG 2D and 3D Graphs.

Examples of real-time charts with integrated scrolling and cursor feedback may be found in the Real-Time Strip-Chart Demo.

This example uses only some of the resources available in any GLG graph and only a small number of possible settings for these resources. This demo applet generates simulated data for updating the graph. In an application, the data can be obtained from any Web data source.

The Applet Source Code may be used as an example of using resources of GLG 2D and 3D graph widgets.

Notice: Script buttons may not work in some browsers due to the LiveConnect's new security settings.
