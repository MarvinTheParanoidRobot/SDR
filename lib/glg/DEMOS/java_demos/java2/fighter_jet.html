3D Fighter Jet Visualization

This a Java version of the F-35 Lightning II Demo that visualizes the fighter jet in 3D and animates its movement. This type of 3D visualizations really benefits from using OpenGL, which is not available in a Java applet. To use OpenGL with hardware-accelerated rendering, please download GLG Binary Demos.

The drawing contains sliders that control Roll, Pitch and Yaw of the aircraft, as well as its scale, position and update speed. The Wire Frame toggle may be used to switch between Fill and Wire Frame modes. On startup, the aircraft is updated with simulated data until the Roll, Picth or Yaw sliders are moved to control the jet manually. To restart the simulated update mode, increase the update speed using the Update Speed slider.
