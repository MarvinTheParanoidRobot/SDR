    Overview

    This demo demonstrates the features of the GLG Toolkit often used in the Network Monitoring applications:

  • Vector or Image map display (see the GIS Demo for GIS maps)
  • Dynamic node and link icons
  • Changing node and link icon attributes to display their status
  • Scrolling alarm list
  • Layering and selective display of cities and map layers.
  • Moving nodes and links and creating new nodes and links
       Use toolbar buttons to select the mouse use modes, use tooltips to identify the buttons in the toolbar.
  • Selecting nodes to display node load history in a graph
  • Zooming and panning
    •

    The drawing was created using the GLG Graphics Builder. The demo uses similated data. However, provided with a data source, it can display live data.

    User Interface

  • Use sliders to zoom, pan and rotate the drawing. The scrollbars on the sides of the drawing scroll the drawing in 2D mode, or rotate the drawing in 3D mode.
  • Use the Big Cities, Medium Cities and Map buttons in the toolbar to toggle layers of objects on the map. Use tooltips to identify toolbar buttons.
  • Use button tooltips as help.
  • Use the Select Node, Move Node, Create Node or Create Link toolbar buttons to switch between different mouse modes.
  • In the Select Node mode, select different nodes with the mouse to display the node's load history as a graph.
  • In the Move Node mode, click and drag the nodes with the mouse.
  • In the Create Node and Create Link modes, add new nodes and links to the map. To add a node, click in the map to position a node. To create a link, use the mouse to select two nodes to be connected with a new link.
