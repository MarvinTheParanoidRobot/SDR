GLG Java Controls Demo

This demo shows an example of GLG controls. All these control were be built with the GLG Graphics Builder and are included in the Controls Widget Set, which also includes many other dials, meters and gauges.

User Interaction

All controls may accept user input. Press the Stop button to stop the supply of the simulated data and try moving controls with the mouse.
