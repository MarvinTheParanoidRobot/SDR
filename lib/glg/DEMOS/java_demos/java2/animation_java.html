GLG Java Animation Example

This "catch me" game whose drawing was created using the GLG Graphics Builder, shows a simple example of a GLG animation. A move dynamics is used to move the ball, and a color dynamics is used to blink the ball when you catch it by clicking on it with the mouse. A Selection Callback is used for hit detection, and an Input Callback handles buttons in the drawing.

User Interaction:

  • To win, you must click on the ball with the mouse.
  • Use the buttons in the drawing to adjust the animation speed.
